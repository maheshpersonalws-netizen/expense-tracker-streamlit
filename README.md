Expense Tracker & Invoice Generator

A simple Streamlit-based Expense Tracker and Invoice Generator
that allows users to record expenses, visualize spending,
and generate invoices in PDF format.

 Features
- Add expenses manually
- Expense visualization using bar charts
- Export data to CSV / Excel
- Generate invoice PDF
- Simple and clean UI

 Tech Stack
- Python
- Streamlit
- Pandas
- Matplotlib
- FPDF

How to Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
